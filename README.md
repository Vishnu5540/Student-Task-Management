# Student Task Management System (Java)

## Description
A console-based Java application that allows users to manage daily tasks.
The project demonstrates core Java concepts, object-oriented programming,
and use of collections.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Simple and user-friendly console interface

## Technologies Used
- Java
- OOP Concepts
- Java Collections (ArrayList)
- Scanner for user input

## Learning Outcomes
- Improved understanding of Java classes and objects
- Hands-on practice with collections
- Strengthened problem-solving skills
